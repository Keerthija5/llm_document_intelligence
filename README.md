# LLM-Based Document Intelligence System

## 📌 Overview

This project focuses on building a **Document Intelligence System using Large Language Models (LLMs)** to process unstructured business text and convert it into structured insights.

The goal is to simulate real-world scenarios such as:

* support tickets
* maintenance reports
* meeting notes

and automatically extract useful information from them.

---

## 🚀 What I Implemented

* End-to-end document processing pipeline
* LLM-based summarization using Hugging Face
* Model comparison (**BART vs T5**)
* Custom evaluation logic to compare model outputs
* Action item extraction from text
* Keyword extraction using NLP preprocessing
* Document classification (category, priority, urgency)
* Structured outputs in **JSON and CSV format**

---

## ⚙️ Workflow

```text
Input Text → Summarization (BART & T5) → NLP Processing → Evaluation → Output (JSON + CSV)
```

---

## 🤖 Model Comparison

| Model    | Observation                                    |
| -------- | ---------------------------------------------- |
| **BART** | More fluent and structured summaries           |
| **T5**   | Slightly longer but less consistent formatting |

The evaluation system compares both models based on:

* sentence completeness
* formatting quality
* readability
* structure consistency
